# threekit-packages

This uses bun. Please install bun from here: https://bun.sh

## Developer Setup

Install dependencies

```bash
bun install
```

Build the project

```bash
bun run build
```

Run the tests

```bash
bun run test
```

Watch for changes and rebuild as necessary, while starting both the demo-app and the demo-server in dev mode.

```bash
bun run watch
```
