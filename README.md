# Development

### Steps for start Frontend part 
```
npm install
npm run dev
```

### Steps for start Backend part 
```
cd server
npm install
npm start:dev
```

### Steps for starting a project
- start backend
- start frontend

### Branches
- main (for production)
- staging (for development)

### Architecture
 - UML diagrams, located in the docs folder (use StarUML)

# Production

### Steps

docker-compose up --build -d
